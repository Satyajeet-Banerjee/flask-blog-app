# Flask Blog App

A full-featured blog application built using Flask.

## Features

- User Registration & Login
- Create, Edit, Delete Blog Posts (Admin only)
- Comment system
- Contact form (Email integration)
- Gravatar profile images

## Tech Stack

- Flask
- SQLAlchemy
- Flask-Login
- Flask-WTF
- Flask-Mail

## Setup Instructions

1. Clone the repository:
git clone https://github.com/Satyajeet-Banerjee/flask-blog-app.git

cd flask-blog-app


2. Create virtual environment:
python -m venv .venv
.venv\Scripts\activate


3. Install dependencies:
pip install -r requirements.txt

4. Create `.env` file:

Use `.env.example` as reference.

5. Run the app:
python run.py
